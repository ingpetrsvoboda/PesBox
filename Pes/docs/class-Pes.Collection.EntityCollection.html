# getByIdentity( Pes\Entity\Persistable\IdentityInterface $identity )

Vrací člen kolekce se zadanou iodentitou. Pro rozpoznání člena kolekce je rozhodující shoda objektu identity, nikoli hodnoty vlastností objektu identity. Shoda objektu identity musí být úplná, musí se jednat o tentýž objekt identity, který měl člen kolekce při svém vložení do kolekce. Pokud byl objekt vložen v okamžiku, kdy neměl identitu a identita objektu byla nastavena později (např. při persistování objektu) nelze takový objekt v kolekci touto metodou nalézt.
