# sort( callable $callback )

Metoda seřadí prvky kolekce s pouřitím porovnávací funkce zadané jako parametr. Porovnávací funkce musí porovnávat členy kolekce, vracet při volání callback(první, druhý) tyto hodnoty: 1 pokud 'první' má být před 'druhý', 0 pokud je pořadí členů stejné, -1 pokud má být 'druhý' před 'první'. Metoda interně používá metodu uasort, pro porobnosti o porovnávací funkci vit dokumentace php. Po použití metody iterování kolekce (foreach) probíhá v stříděném pořadí, pozor - indexy členů kolekce se tříděním nijak nemění.
