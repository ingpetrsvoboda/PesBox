Konstruktor. S pomocí instanční proměnné dsn provider sestaví řetězec DSN a s parametry user, password volá konstruktor rodiče - PDO. Pokud je nastavena instanční proměnná attributes setter, pak s použitím tohoto objektu nastaví vytvořenému objektu handleru atributy (atributy PDO). Bezpečnostní rizika: Objekt handler má jako své vlastnosti uložen objekt dsnProvider, která obsahuje informace i připojení a vrací ho metodou getDsnProvider(). Jedná se současně o menší bezpečnostní riziko. Dále má objekt handler uloženo jméno uživatele, se kterým se připojoval k databázi a vrací ho metodou getUser(). Zde se jedná o větší menší bezpečnostní riziko. Objekt handler neobsahuje hodnotu parametru pass (heslo uživatele, pod kterým se připojoval k databázi). Kombinace dsn, jména a hesla by již představovala velké bezpečnostní riziko. Objekt se používá obvykle na mnoha místech v rámci celé aplikacea neůze vyloučit, že bude někde zobrazování, vypisována jeho hodnota, bude logován apod. a je tedy třeba mít uvedená rizika na paměti.
