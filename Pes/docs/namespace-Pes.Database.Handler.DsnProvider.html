Interfaces summary
DsnProviderInterface

Vrací dns řetězec pro db handler. Je připraven pouze na dsn pro připojení k databázi prostřednictvím specializovaného db handeru, např. mysql, sqlsrv apod. Není připraven na připojení k socketu (bez use, password) ne na připojení přes ODBC (výrazně odlišná skladba dsn). Předpokládá, že pro dsn byly potřebné parametry jméno databáze a host, případně port a kódování. Předepisuje metody pro vracené takových parametrů.
