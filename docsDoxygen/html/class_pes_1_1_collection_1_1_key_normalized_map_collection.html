Detailed Description

NormalizedKeyMapCollection

Kolekce párů klíč-hodnota. Rozšiřuje funkčnost MapCollection o normalizaci klíčů. Pokud je nastaven normalizer, hodnoty v kolekci jsou interně ukládány podle normalizovaného klíče. Pro vytváření normalizovaé podoby klíče se používá normalizer - objekt KeyNormalizatorInterface, který je nastaven metodou setKeyNormalizer().

Možné použití je např. pro case insensitive klíče - zadaný objekt keyNormalizer musí převádět klíč na lowercase (nebo uppercase) variantu.
