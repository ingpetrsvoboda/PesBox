Detailed Description

Description of LikeMatcher Ověřuje zda hodnota odpovídá vzoru obdobně jako SQL operátor LIKE. Používá stejné wildcard znaky ('_' a '*') a akceptuje escape znak '\' pro escapování znaků '' a '_', které nemají být chápány jako wildcard znaky. Escape znak je defaultně '\' a může být zadán jiný pomocí parametru konstruktoru. Je shodně s SQL lIKE case insensitive.
