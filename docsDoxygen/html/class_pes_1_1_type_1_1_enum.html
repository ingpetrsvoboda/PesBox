Detailed Description

Emuluje typ Enum (obdobně jako SplEnum)

SplEnum: This » PECL extension is not bundled with PHP.

Použití - definice typu

namespace Framework; use Framework; class DbType extends Enum { * const MySQL = 'mysql'; const MSSQL = 'mssql'; }

Vytvoření proměnné

try { $dbType = new DbType(); $msType = $dbType('mssql') //OK, vrací řetězec 'mssql' $blaType = $dbType('bla'); // Vyhodí výjimku - za běhu } catch (Pes $notInEnumExc) { echo $notInEnumExc->getMessage() . PHP_EOL; }

Bezpečné vytvoření proměnné (bez rizika vyhozené výjimky) - použití konstanty třídy

$dbType = new DbType(); $msType = $dbType(DbType::MSSQL) //OK, vrací hodnotu 'mssql'

Test "if"

//call: $dbType = new DbType(); $someObject->someMethod($dbType, DbType::MySQL); //test: function someMethod(DbType $enum, $value) { if ($value == $enum($value)) { }; }
